Foundational Java Algorithms Website

Description

The Amazing Button Masher Website

Description

A simple game website where the premise is to mash the same button over and over again to increase your score. The website implements Firebase's Realtime Database in order to allow users to create and authenticate their own personal accounts that keeps track of their progress.

There is a leaderboard system implemented as well that tracks the top five users with the highest scores and displays their username and score in the top left corner of the game screen. There is also a store section where users can unlock different buttons to use after hitting certain milestones.

Technologies Used

HTML, CSS, JavaScript, Firebase Realtime Database

Super Awesome Canvas Drawer Website

Description

HTML based website that can create drawing canvases with varying levels of pixel density. Each pixel is a blank div element whose size and positioning on the canvas is determined by implemented CSS grids. The website also comes with features such as a brush tool, an eraser tool, a fill tool, and a start over option.

This website also takes advantage of DOM manipulation and the HTML Canvas library to save the canvas state as a downloadable image.

Technologies Used

HTML, CSS, JavaScript

Cow Tic-Tac-Toe

Description

A game fully created with Unity's 2D game engine that replicates the well-known Tic-Tac-Toe game with three different levels. Each level increases the board size and even has obstacles in the final level. The gameplay script is written in C#.

The game is styled with cute cartoon cow and bunny characters residing in a created rural landscape with lakes and forests. All sprites and textures were purchased online. There is also a created start screen and end screen.

Technologies Used

Unity Game Engine, C#

UCF'S Declassified Guide to Commiting Tax Fraud

Description

A project created for a Hackathon with two other members. The premise of the project is a game in which a man who is commiting tax fruad has a nightmare in which he is chased by former Commissioner of the IRS Charles Rettig.

The main gameplay is in the nightmare sequence in which the controllable character participates in an unless running-style game in which he must avoid obstacles while being chased. When the player is eventually caught, he is forced to fill out a 1040 tax form.

This project was my first exposure to the Godot game engine. My role in the project was focused more on the sprite designs and animations as well as the start screen, bedroom, and opening cut-scene.

Technologies Used

Godot Game Engine, GDScript, Aseprite
